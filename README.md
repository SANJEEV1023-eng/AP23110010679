# Frontend Logging System

## Description
This project sends logs from frontend to evaluation server using REST API.

## Setup Instructions

1. Install dependencies
npm install

2. Add your token
Create .env file:
VITE_TOKEN=your_token_here

3. Run project
npm run dev

4. Open browser
http://localhost:5173

## Features
- Log API integration
- Error handling
- Button actions logging

## Tech Stack
- React (Vite)
- Axios
